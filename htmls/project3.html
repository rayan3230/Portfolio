Project 3: University Scheduler

Project Description

This is a desktop application developed in Java designed to manage university schedules for both students and administrators.

Key Features:

  • Separate interfaces for Student and Admin roles
  • Student View: Check personal schedules, view available courses
  • Admin View: Manage courses, assign schedules, oversee student registrations
  • Database integration for persistent storage of schedules and user data
  • Login functionality to ensure secure access

Technologies Used:

  • Java
  • SQL/Database Management
